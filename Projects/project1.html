PROJECT 01

MATLAB CFD Solver for 2D Airfoils

MATLAB CFD URANS-SA Airfoil Analysis
Why This Project?

Performing a complete CFD analysis of an airfoil is often time-consuming and fragmented, especially for students who must switch between different tools for geometry creation, meshing, solver execution, and visualization. Each step requires separate scripts or software, making it difficult to track how early decisions influence the final aerodynamic results.

Commercial CFD software tends to hide matrix assembly, turbulence-model behavior, and residual evolution behind automated presets. Users can generate results quickly, but they rarely gain insight into how the solver works internally. This project bridges this gap by developing a complete end-to-end CFD environment inside MATLAB.

How Was It Done?

The project was executed through the development of six integrated modules:

  • Geometry Module: Automated airfoil generation using NACA 4-digit definitions with smooth curvature representation.
  • Mesh Module: Delaunay triangulation with distance-based sizing function and near-wall refinement for boundary layer capturing.
  • Solver Module: Finite-element URANS formulation with Spalart-Allmaras turbulence modeling for accurate flow prediction.
  • Post-Processing Module: Visualization of pressure/velocity contours, streamlines, and residual histories.
  • Validation Module: Comparison of MATLAB solver outputs with ANSYS Fluent results.
  • GUI Module: User-friendly interface managing the entire CFD pipeline without requiring coding.
